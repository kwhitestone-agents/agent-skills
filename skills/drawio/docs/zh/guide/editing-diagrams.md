# 编辑图表

学习如何修改和优化现有图表。

## 编辑操作

### 修改节点属性

更改特定节点的外观或内容：

> "将'处理数据'节点的背景颜色改为浅蓝色"

> "为所有决策节点添加粗边框"

### 添加节点

在图表中插入新元素：

> "在'支付失败'节点后添加一个'记录错误'节点"

> "在'验证输入'和'处理数据'之间插入一个决策节点，检查输入是否有效"

### 连接节点

在节点之间创建边：

> "用箭头连接'开始'节点到'验证输入'节点"

> "添加从'数据库'到'缓存'的曲线连接，标签为'sync'"

### 删除元素

删除节点或连接：

> "删除'临时存储'节点及其连接"

## 高级编辑

### 批量操作

对多个元素应用更改：

> "将所有处理节点的背景设为浅灰色"

> "为所有矩形节点添加圆角"

### 布局调整

重新组织图表：

> "将节点排列为水平布局"

> "为服务节点创建网格布局，分为 3 列"

### 样式优化

精细调整视觉外观：

> "使连接线更粗，并使用深灰色"

> "为所有节点添加阴影"

## 使用单元格 ID

要进行精确编辑，可以按 ID 引用特定单元格：

1. 让 Claude 显示单元格 ID：
   > "显示此图表中的单元格 ID"

2. 使用 ID 编辑：
   > "将单元格 J5 的文本更改为'新流程名称'"

3. 按 ID 修改属性：
   > "将单元格 J3 的填充颜色设置为 #FF6B6B"

## 有效编辑的技巧

1. **描述清晰**：清楚地说明你想要更改的内容
2. **引用上下文**：使用附近的节点作为参考点
3. **迭代**：进行小更改并查看结果
4. **使用预览**：每次更改后检查浏览器预览
