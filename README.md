# Agent Skills

个人使用的 agent skills 集合。

## Skills

- `dev-browser` - 浏览器自动化
- `drawio` - Draw.io 图表操作
- `frontend-design` - 前端界面设计
- `mcp-builder` - MCP 服务器构建
- `pptx` - PowerPoint 演示文稿处理
- `skill-creator` - 技能创建工具
- `ui-ux-pro-max` - UI/UX 设计

## 安装

运行 `setup-links.bat` 创建符号链接到 Copilot 和 Claude 配置目录。
